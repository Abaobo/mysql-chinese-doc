# mysql 中文翻译文档

## 创建过程

首先你要确认你的系统中存在`nodejs`和`npm`.

然后执行以下命令安装环境

```bash
npm install gitbook -g
```

部署环境：

```bash
git clone https://github.com/mowangjuanzi/mysql-chinese-doc.git
cd mysql-chinese-doc
gitbook install
gitbook serve
```
