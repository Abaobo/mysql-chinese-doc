简介

包含 MySQL NDB Cluster 8.0

摘要

这是 MySQL™ 参考手册. 它记录了从 MySQL 8.0 到 8.0.17, 以及分别基于从 8.0 到 8.0.16-ndb-8.0.16 的 NDB 集群版本. 它可能包含尚未发布的 MySQL 版本特性的文档. 有关已发布的版本的信息, 请参阅 MySQL 8.0 发布说明.

MySQL 8.0 特性. 这个手册描述的功能可能不包含在 MySQL 8.0 版本中; 这些功能可能不包含在授权给你的 MySQL 8.0 版本中. 如果你对 MySQL 8.0 版本中包含的特性有任何疑问, 请参考您的 MySQL 8.0 l许可协议或联系您的 Oracle 销售代表.

有关每个版本的详细修改说明, 请参阅 MySQL 8.0 发布说明.

有关法律信息, 包含许可信息, 请参阅 前言和法律条款.

有关使用 MySQL 的帮助, 请访问 MySQL 论坛 或者 MySQL 邮件列表, 你可以在那里与其他 MySQL用户讨论你的问题.
